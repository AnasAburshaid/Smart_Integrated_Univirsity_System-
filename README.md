# 🎓 Smart Integrated University Portal System

> A comprehensive, three-tier academic and administrative management platform built for modern university ecosystems.

## 📖 Overview
This **University Portal System** is designed to streamline academic and administrative services. The system provides **three separate interfaces** tailored specifically for **Students, Teachers, and Administrators**, ensuring secure and role-specific access.

The project was developed as part of the **Bachelor in Software Engineering** at the **University of Jordan**, supervised by **Dr. Khalid Alkharabsheh**.

---

## 🖥️ My Role: Backend Development
As part of the engineering team, I architected and implemented the core backend logic, focusing on:
*   **🔐 User Authentication & Role Verification:** Ensuring secure access across different user tiers.
*   **⚙️ API Request Processing:** Handling distinct operational logic for Students, Teachers, and Admins.
*   **📚 Academic Management:** Processing courses, grading workflows, and system announcements.
*   **🗄️ Database Integration:** Structuring a decoupled SQLite database for high reliability and data persistence.
*   **🌐 RESTful API Endpoints:** Built and tested using the Drogon Framework for seamless frontend communication.

---

## ✨ Key Features by Role

### 👨‍🎓 Student Interface
*   **Profile Management:** View and update personal information.
*   **Course Enrollment:** Register and drop courses seamlessly.
*   **Academic Tracking:** Monitor grades and check upcoming exam schedules.
*   **Campus Updates:** Access course and system-wide announcements.

### 👨‍🏫 Teacher Interface
*   **Roster Management:** View assigned courses and enrolled student lists.
*   **Grading System:** Add, update, and manage student grades securely.
*   **Class Communication:** Create and publish course-specific announcements.

### ⚙️ Administrator Interface
*   **User Management:** Full CRUD operations (add, update, delete) for all user types.
*   **Curriculum Control:** Manage courses and assign faculty to classes.
*   **Term Scheduling:** Organize academic semesters and overarching exam schedules.
*   **Global Announcements:** Publish critical system-wide updates.
