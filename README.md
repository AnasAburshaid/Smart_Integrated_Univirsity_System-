## Overview
This is a **comprehensive University Portal System** designed to manage academic and administrative services within a university environment.  
The system provides **three separate UIs** for **Students, Teachers, and Administrators**, each tailored to the user‚Äôs role.  
The backend is built in **C++** using the **Drogon API Framework**, with a **SQLite** database, and the frontend is built using **React.js, TypeScript, and Vite**.  

This project was developed as part of the **Bachelor in Software Engineering** at Al-Balqa Applied University, supervised by **Dr. Khalid Alkharabsheh**.

## Features

### Student Interface
- View and update profile information  
- Register and drop courses  
- View grades and exam schedules  
- Access announcements  

### Teacher Interface
- View assigned courses and student lists  
- Add and update student grades  
- Create and manage course announcements  

### Administrator Interface
- Manage users (add, update, delete)  
- Manage courses and course assignments  
- Manage academic semesters and exam schedules  
- Publish system-wide announcements

## Tech Stack
- **Backend:** C++ with Drogon API Framework  
- **Frontend:** React.js, TypeScript, Vite  
- **Database:** SQLite
